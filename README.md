# Simple-paint
This is a simple paint program using the OOP technique for learning purposes.

The saved file contains information about the attributes of the shape objects.


The extension of the saved file is "*.â€‹shapeobj".
## Features:

* Basic: New, Open, Save, Save as.
* Undo: undo the last change.
* Redo: redo the last change.
* Shapes: line, rectangle, triangle, ellipse.
* Color: background, foreground.
* Width: brush size.

## Keyboard Shorcuts:

* New: Ctrl + N
* Open: Ctrl + O
* Save: Ctrl + S
* Save as: F12
* Undo: Ctrl + Z
* Redo: Ctrl + Y
